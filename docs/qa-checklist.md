# QA Checklist

1. 첫 실행에서 장르 카드 선택 후 스튜디오로 진입한다.
2. 재생 버튼을 눌러 30초 안에 첫 소리가 난다.
3. `+ 사운드`로 루프를 추가한다.
4. 루프 카드 필터가 카테고리/장르별로 동작한다.
5. 클립을 좌우로 드래그하면 1/4마디 단위로 붙는다.
6. 클립 오른쪽 끝을 드래그하면 길이가 바뀐다.
7. 클립 메뉴에서 복제와 삭제가 동작한다.
8. 실행취소/다시실행이 편집을 되돌린다.
9. BPM 슬라이더 변경이 재생 중 반영된다.
10. 메트로놈 켜기/끄기가 동작한다.
11. 악기 탭에서 드럼패드가 소리를 낸다.
12. 악기 탭에서 스케일 키보드가 소리를 낸다.
13. 연주 녹음 후 클립으로 넣기가 동작한다.
14. 마이크 권한 허용 후 녹음 클립이 추가된다.
15. 마이크 권한 거부 시 앱이 멈추지 않는다.
16. 믹서에서 소리 크기와 좌우 위치가 바뀐다.
17. 솔로 2개를 켜면 해당 트랙만 들린다.
18. 이펙트 시트에서 공간감/에코/톤 설정이 저장된다.
19. 새로고침 후 프로젝트가 IndexedDB에서 복원된다.
20. WAV 내보내기 파일이 생성된다.
