# 번들 루프 명세

MVP에는 저작권 문제가 없는 합성 WAV 루프 48개를 포함합니다. 모든 루프는 90 BPM 기준으로 생성되며, 프로젝트 BPM이 바뀌면 `playbackRate`로 따라갑니다.

| 카테고리 | 개수 | 길이 | 역할 |
|---|---:|---:|---|
| 드럼 | 16 | 1마디 | 킥, 스네어, 하이햇 중심의 스타터 그루브 |
| 베이스 | 12 | 2마디 | Cm/Am/F 계열의 낮은 루프 |
| 멜로디 | 12 | 2마디 | 입문자가 얹기 쉬운 짧은 리프 |
| 효과음 | 8 | 1마디 | 전환, 라이저, 노이즈 질감 |

파일 위치: `public/loops/*.wav`

생성 방법: `npm run generate:loops`

향후 실제 샘플로 교체할 때는 CC0 또는 상업적 사용 가능한 라이선스만 사용하고, 원본 출처와 라이선스를 이 문서에 추가합니다.
